1.CREAR NUESTROS DOCKERFILES 
  1.1 DOCKERFILE BUILD PHP
  1.2 DOCKERFILE BUILD MYSQL
2. PUSH A NUESTRAS IMAGENES
  2.1 DOCKER PUSH MY-DOCKER-PHP
  2.2 DOCKER PUSH MY-DOCKER-MYSQL
3.ENCENDER MINIKUBE
  3.1 MINIKUBE START
4. CREAR NEUSTRO NAMESPACE
  4.1 KUBECTL APPLY -F NOMBREARCHIVO.YAML 
  4.2 KUBECTL GET NAMESPACE
5. EJECUTAR NUESTROS DEPLOYMENT 
  5.1 KUBECTL APPLY -F DEPLOYNAME.YAML --NAMESPACE=PRACTICAFINAL
  5.2 KUBECTL GET PODS -N=PRACTICAFINAL
6. EJECUTAR LOS SERVICIOS 
  6.1 KUBECTL APPLY -F SVC-NAME.YAML --N=PRACTICAFINAL
  6.2 KUBECTL GET SVC -N=PRACTICAFINAL
7. EJECUTAR LOS VOLUMES
  7.1 KUBECTL APPLY -F VOL-NAME.YAML --N=PRACTICAFINAL 
  7.2 KUBECTL GET PV -N=PRACTICAFINAL
  7.3 KUBECTL GET PVC -N=PRACTICAFINAL
8. EJECUTAR EL INGRESS
  8.1 KUBECTL GET INGRESS -N=PRACTICAFINAL
